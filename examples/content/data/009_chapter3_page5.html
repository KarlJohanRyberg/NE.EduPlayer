quiz

Quiz is used to ask questions to the user and collect the response and give feedback.

Note!
At the moment the quiz component only support one question. It will render more than one quistion but the logic for submitting and giving feedback för multiple quistions are not implemented 

<div class="NE-plugin-container" data-plugin="quiz" data-settings="{
    'datafile': '../content/data/quizes/quiz-01.txt', // Path to the quiz data file
    'ID':'quiz-example' // The desierd ID of the component
    }">
</div>

Events: OnSubmit
<script type="text/javascript">
    NE.Plugin.AttachEvent('quiz-example', 'OnSubmit', function (sender) {
        // Custom code goes here
    });
</script>
