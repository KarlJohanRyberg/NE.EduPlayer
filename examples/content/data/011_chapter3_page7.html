verticalnav

Verticalnav is used to display e series of sub pages in a vertical slider. With or without visible navigation. It can be used as a way for the user to browse or as a way to make selection.
With the filter function and no visible navigation, it can be used to change page content based on user selections. 

<div class="NE-plugin-container" data-plugin="verticalnav" data-settings="{
    'ID': 'Vert_Example', // The desierd ID of the component
    'hidenavigation': 'false', // If set to true, the navigation is not displayed in the panel
    'pages': [ // A list of page objects to display in the panel
    {
        'datafile': '../content/data/vert_example/opt-1-0.html', // Path to the page data file
        'group':'0' // A group index of the page, is used by the Filter function
    },
    {
        'datafile': '../content/data/vert_example/opt-1-1.html',
        'group':'0'
    }
    ]
}">
</div>

Events: OnChanged
<script type="text/javascript">
    NE.Plugin.AttachEvent('Vert_Example', 'OnChanged', function (sender) {
        // Custom code goes here
    });
</script>

Methods: Filter
    NE.Plugin['Vert_Example'].Filter(groupIndex, pageIndex);
    // Hides all pages in the verticalnav panel except the ones 
    // matched by the group index and the optional page index
