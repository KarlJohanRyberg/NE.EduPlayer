Components

To use a component, add a DIV-tag with the class name "NE-plugin-container" and the name of the component in the "data-plugin" attribute. Here are some esxamples of the different components and a brief explanation og the settings of each components

Note that if you supply an ID to a components in its settings property, it will accessable from javascript through NE.Plugin['ID'].

chapterlist

The "chapterlist" component displays a list of all chapters in the course that has the "displayInMenu" property set to true. 

<div class="NE-plugin-container" data-plugin="chapterlist" data-settings="{
                    'ID': '', // The desierd ID of the component
                    'linkPrefix': '', // The chapter links will get an ID with this prexis plus an index
                    'panelClasses': '', // Extra CSS classes that shall be added to the component
                    'breaklimit': 2 // The number of links per row
                    }">
</div>

multiplechoice

The "multiplechoice" component shows a list of panels where the user can pick one. 

<div class="NE-plugin-container" data-plugin="multiplechoice" data-settings="{
            'ID': // The desierd ID of the component
            'SelectOne': // Set to false if the user should be able to pick more than one option
            'OptionIDPrefix': // The panels will get an ID with this prexis plus an index
            'options': [ // A list of the different options, each will render oen panel
                        {
                            'Images': ['default.xx','hover.xx','active.xx'], // A list of images for the different states
                            'Title': '', // The title of the panel
                            'Text': // Panel text
                        }
            ]
        }">
</div>
Events: OnChanged
<script type="text/javascript">
    NE.Plugin.AttachEvent('component_ID', 'OnChanged', function (i_items) {
        // Custom code goes here
    });
</script>

nextbutton

This comonent shows a button that will take you to the next page in the course 

<div class="NE-plugin-container" data-plugin="nextbutton" data-settings="{
                    'ID': '', // The desierd ID of the component
                    'text': '', // The text of the button
                    'initClasses': '', // Extra CSS classes that shall be added to the button
                    'iconclass': '' // The Font Awesome icon of the button. Defaults to 'fa-angle-down'
                    }">
</div>
Events: ClickOverride
<script type="text/javascript">
    NE.Plugin.AttachEvent('component_ID', 'ClickOverride', function (sender, e, base) {
        // Custom code goes here
        // If you want to call the buttons default callback, call base()
        base(sender, e);
    });
</script>

quiz

Quiz is used to ask questions to the user and collect the response and give feedback.

revealbutton

Revealbutton is used to toggle the visibility of a part of the page.

verticalnav

Verticalnav is used to display e series of sub pages in a vertical slider. With or without visible navigation. It can be used as a way for the user to browse or as a way to make selection.
With the filter function and no visible navigation, it can be used to change page content based on user selections.
