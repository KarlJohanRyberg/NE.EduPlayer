Common CSS classes

In this chapter I will go through some of the CSS classes that can be nice to have.

Layout & panels

NE-full-width

The class "NE-full-width" is used to make an area expand to the edges of the browser window. Often used togeter with dividers and different backgground colors to create visual separation.

"NE-full-width" have to be located outside of any "container" element. It can itself contain a "container" to layout it's content, as an ordinary Bootstrap container

There are a some of pre-defined color classes for setting foreground and background colors of panels and text elemets.
See a couple of examples below.

NE-full-width

NE-info-panel

Use "NE-info-panel" to visually extract content from the flow of the page. Often used for example texts

NE-checklist-panel

"NE-checklist-panel" is used to present checklists to the users.
